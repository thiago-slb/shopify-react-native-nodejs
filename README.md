# Shopify React Native

React Native shopping proof of concept with an Expo mobile app and a Fastify backend that mediates
Shopify Storefront API access.

## Project Docs

- [App README](app/README.md) - Expo app setup, environment, flows, and frontend architecture.
- [Server README](server/README.md) - Fastify backend setup, Shopify integration, API contract, and backend flows.

## Repository Layout

```text
/app      Expo React Native app
/server   Fastify Shopify integration backend
```

Start with the server README if you need Shopify/backend configuration, then the app README to run
the mobile client against it.
